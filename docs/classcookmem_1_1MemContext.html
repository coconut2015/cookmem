Detailed Description

template<class Arena, class Logger, class T = void>
class cookmem::MemContext< Arena, Logger, T >

A basic memory context that creates a MemPool based on the arena and logger references user passed in.

This class is mostly used in cases when arena or logger that do not have default constructors.
