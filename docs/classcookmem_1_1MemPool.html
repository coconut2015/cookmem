Detailed Description

template<class MemArena, class T = void>
class cookmem::MemPool< MemArena, T >

A memory allocator.

Note that this class only provides the algorithm necessary to deal with small memory chunks. The actual ability to get memory segment (such as mmap memory pages) is deferred to memory arena.
